# Produtos e Serviços — Axon Labs

> Catálogo que o copiloto usa para sugerir a oferta certa ao vendedor.
> **Valores sugeridos com base no mercado brasileiro de 2026** (agências cobram
> tipicamente R$ 500–2.000/mês + setup de R$ 500–5.000). Ajuste conforme o seu
> custo real de IA/API e o seu posicionamento. Todos os valores são "a partir de",
> porque o escopo varia por cliente.

## 1. Agente de Atendimento no WhatsApp

Chatbot com IA que responde clientes 24/7 no WhatsApp, treinado com o conhecimento
do próprio negócio (produtos, FAQ, tom de voz). Passa a conversa para um humano
quando o caso exige.

- **Resolve:** demora na resposta, lead que esfria fora do horário, atendente
  sobrecarregado com perguntas repetidas.
- **Ideal para:** PMEs de serviço e comércio que vivem no WhatsApp.
- **Entrega:** agente + base de conhecimento + handoff para humano.
- **Investimento sugerido:** setup a partir de **R$ 2.900** + mensalidade a partir
  de **R$ 690/mês** (manutenção, ajustes e custo de IA/API).

## 2. Automação de Vendas & CRM

Centralização de leads no Airtable com automações em Make/n8n: captura pelo
formulário inteligente, registro automático no CRM, follow-up e notificações
para o time.

- **Resolve:** lead anotado em papel/planilha solta, follow-up esquecido,
  falta de visão do funil.
- **Ideal para:** PMEs e e-commerces que já geram leads mas perdem no
  acompanhamento.
- **Entrega:** CRM no Airtable + fluxos no Make/n8n + integração via webhook.
- **Investimento sugerido:** setup a partir de **R$ 3.500** (projeto único);
  manutenção opcional a partir de **R$ 490/mês**.

## 3. Automação de E-commerce

Fluxos para lojas online: recuperação de carrinho abandonado, suporte
automatizado, respostas de pós-venda (rastreio, troca, status do pedido).

- **Resolve:** carrinho abandonado sem recuperação, suporte lento, time de
  atendimento afogado no pós-venda.
- **Ideal para:** e-commerces com volume de mensagens repetitivas.
- **Entrega:** agente de suporte + automações conectadas à plataforma da loja.
- **Investimento sugerido:** setup a partir de **R$ 3.900** + mensalidade a partir
  de **R$ 890/mês** (volume maior de conversas + recuperação de carrinho).

## 4. Projeto Sob Medida (n8n / Make)

Automação personalizada para um fluxo específico do cliente (ex.: conectar
sistema X ao WhatsApp, gerar relatórios automáticos, integrar ferramentas que
não conversam entre si).

- **Resolve:** trabalho manual repetitivo e sistemas que não se integram.
- **Ideal para:** negócios com um gargalo operacional bem definido.
- **Entrega:** escopo fechado após diagnóstico.
- **Investimento sugerido:** projetos a partir de **R$ 4.500**, sob orçamento
  após a call de diagnóstico.

## Como o copiloto deve recomendar

1. Primeiro entender a **dor** (demora? follow-up? pós-venda? integração?).
2. Mapear a dor para **um** serviço principal — não empurrar tudo de uma vez.
3. Sugerir o próximo passo natural: **call de diagnóstico gratuita**.
4. Nunca confirmar valor fechado — sempre "a partir de", com o preço final
   definido na proposta após o diagnóstico.
