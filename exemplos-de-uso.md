# Exemplos de Uso — Prompts de Tarefa

> Prompts curtos que o vendedor manda ao copiloto no dia a dia.
> Todos assumem o `system-prompt.md` ativo e a pasta `knowledge/` anexada.

## 1. Estruturar um pitch

```
Prospect: dona de uma loja de roupas com Instagram forte. Vende muito por DM
mas demora horas pra responder e perde venda de madrugada. Me ajuda a montar
o pitch.
```

## 2. Contornar uma objeção

```
O cliente disse: "achei interessante mas tá caro pro meu momento". Como eu
respondo sem dar desconto na hora?
```

## 3. Escrever um follow-up

```
Fiz uma call de diagnóstico há 4 dias, o cara sumiu. Me escreve um follow-up
leve pra reaquecer sem parecer cobrança.
```

## 4. Qualificar / priorizar um lead

```
Lead novo: quer "uma IA que venda sozinha e demita meus 2 atendentes", não
tem site nem funil, e já perguntou 3 vezes se tem plano de R$ 50. Vale investir
energia?
```

## 5. Adaptar oferta para um segmento

```
Prospect é e-commerce de suplementos com muito ticket de "cadê meu pedido".
Qual serviço eu ofereço e como explico o ganho pra ele?
```

## 6. Preparar para uma reunião

```
Tenho call amanhã com um restaurante que recebe pedido por WhatsApp e vive
perdendo mensagem no horário de pico. Me dá os 3 pontos que eu não posso
esquecer de levantar.
```
